MechWarrior2: 31st Century Combat Codes

Code Description
Press the following keyboard sequence and hold while entering in the code:

CTRL+ALT+SHIFT
CIA Infinite ammo.
BLORB Invulnerability.
COLDMEISER Heat tracking off.
TLOFLYGIRL Add jump jets.
MIGHTYMOUSE Infinite jump jets.
GANKEM Destroys target.
ICANTHACKIT Ends current mission.
MEEPMEEP Time compression enabled.
UNMEEPMEEP Time compression disabled.
HANGAROUND Extended time.
FREEBIRTHTOAD Level warp to select random missions.

MechWarrior2: Ghost Bear's Legacy Codes

Code Description
Press the following keyboard sequence and hold while entering in the code:

CTRL+ALT+SHIFT
PALEX Destroy targeted 'Mech.
KABOOM Kill all units in a mission.
CLARK X-Ray on.
KENT Invincibility.
BURR Disable heat tracking.
JUMBO Jump jets on.
PUTZ End mission successfully.

And some additional Ghost Bear's Legacy codes...

Location Action
